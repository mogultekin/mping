# mping
Multiple Ping (Subnet) (Fast) 
<pre>Pings multiple IPs just in seconds.</pre>
<pre>
Usage: mping <CIDR|range>

Example Usage:
# mping 192.168.200.0/24
192.168.200.1   0.48 ms
192.168.200.2   14.8 ms
192.168.200.50  12.8 ms
192.168.200.75  9.8 ms
192.168.200.85  151 ms
192.168.200.110 0.793 ms
192.168.200.200 0.067 ms
192.168.200.252 0.555 ms
192.168.200.254 0.316 ms

# mping 192.168.200.200-254
192.168.200.200 0.091 ms
192.168.200.252 0.51 ms
192.168.200.254 0.42 ms
</pre>
